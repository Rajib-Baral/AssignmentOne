My CSS Skills in Action

I use CSS to create visually appealing, responsive, and interactive designs. Below are practical examples of my work.

Hover-Animated Card
Interactive Card

Hover over me to see a lift effect!

.hover-card {
  transition: transform 0.3s ease, box-shadow 0.3s ease;
}

.hover-card:hover {
  transform: translateY(-10px);
  box-shadow: 0 10px 20px rgba(0, 0, 0, 0.2);
}

This card uses CSS transitions to create a smooth hover effect, enhancing user interaction.

Responsive Image Gallery
.gallery-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(100px, 1fr));
  gap: 10px;
}

@media (max-width: 576px) {
  .gallery-grid {
    grid-template-columns: repeat(auto-fit, minmax(80px, 1fr));
  }
}

This gallery only has placeholder images. The main focus is to showcase CSS Grid to create a responsive layout that adapts to different screen sizes.

Sticky Navigation Bar
Sticky Navigation Example

Scroll down to see the sticky navigation in action. This section has enough content to demonstrate the effect. CSS makes this possible without JavaScript. 

.sticky-nav {
  position: sticky;
  top: 0;
  background-color: #343a40;
  color: white;
  padding: 10px;
  z-index: 1000;
}

The sticky navigation stays at the top as you scroll, improving user navigation.

Dark Mode Toggle

This section changes between light and dark mode.

:root {
  --bg-color: #ffffff;
  --text-color: #000000;
}

.dark-mode {
  --bg-color: #333333;
  --text-color: #ffffff;
}

.mode-toggle-section {
  background-color: var(--bg-color);
  color: var(--text-color);
  transition: background-color 0.3s ease, color 0.3s ease;
}

This uses CSS variables to toggle between light and dark modes, showcasing theme customization.

Responsive Design with Media Queries

Resize your browser to see this text change!

@media (max-width: 768px) {
  .responsive-text {
    font-size: 18px;
    color: #e74c3c;
  }
}

This text changes size and color on smaller screens, demonstrating responsive design.

CSS Quick Cheat Sheet

  • Font Styling: font-family: 'Roboto', sans-serif;
  • Center Content: margin: 0 auto;
  • Background Color: background-color: #f1f1f1;
  • Text Shadows: text-shadow: 2px 2px 5px gray;
  • Flexbox Centering: 
    display: flex;
justify-content: center;
align-items: center;
