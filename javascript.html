What is JavaScript?

JavaScript is a versatile, high-level programming language used to add interactivity and dynamic content to websites. It runs in the browser and manipulates the DOM.

console.log("Hello, World!");

This logs a message to the console, a basic JavaScript command.

DOM Manipulation

JavaScript interacts with the Document Object Model (DOM) to dynamically update webpage content, styles, and structure.

document.getElementById("myId").innerHTML = "New Content";

This changes the content of an element with the specified ID.

Event Handling

JavaScript enables interactive features by responding to user actions like clicks, keypresses, or mouse movements.

button.addEventListener("click", function() {
  alert("Button clicked!");
});

This adds a click event listener to a button.

Functions and Variables

Functions encapsulate reusable code, and variables store data for dynamic operations in JavaScript.

let count = 0;
function increment() {
  count++;
  console.log(count);
}

This defines a function to increment a counter variable.
